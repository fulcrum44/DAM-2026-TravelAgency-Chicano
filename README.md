 Autor: Joel Chicano López

 ## Instrucciones clonación

 1. En la página principal del repositorio teneemos el boton -Code- para poder acceder al enlace del repositorio.
 2. Copia el enlace https.
 3. En tu equipo como requisito deberás tener instalado XAMPP para poder probar el proyecto.
 4. Una vez instalado XAMPP, abriremos la terminal desde el directorio C:\xampp\htdocs.
 5. Abierta la terminal ejecuta git clone -enlace https del repostirio-
 6. Abriremos nuestro proyecto desde nuestro IDE elegido y ejecutaremos el proyecto con XAMPP iniciado con los servicios Apache y MYSQL iniciados.

## Credenciales administrador
A la hora de probar la página web, en la carpeta sql están las sentencias mínimas para crear las bases de datos y poner datos de prueba.
La tabla de usuarios no tiene datos de prueba. Para probar el usuario administrador, tan solo con crear un usuario llamado admin es más que suficiente.
